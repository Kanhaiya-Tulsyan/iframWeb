Iframe Selector Playground

Dedicated page for testing advanced selectors involving iframes (>>>) only.

Test Case 1 & 2: Checkout Iframe

Example selectors:
iframe#checkout >>> button-pay-now
iframe#checkout >>> #cart .confirm-payment

Inside this iframe you can test: direct element matching and nested elements inside iframe document.

Test Case 3: Site Header Iframe

Example selector:
iframe#site-header >>> .mini-cart-icon

Simple iframe selector where the final CSS selector matches an element inside the iframe document.
